# Number Utility Tool

A simple JavaScript project that analyzes a number and displays:

- Positive, negative, or zero
- Even or odd
- Square of the number
- Factorial
- Prime number check

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download or clone the repository
2. Open `index.html` in a browser
3. Enter a number and click \*\*Analyze Number`
